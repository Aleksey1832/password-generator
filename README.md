# 🔐 Artem Password Pro

Профессиональный генератор безопасных паролей с графическим интерфейсом на Python.

## ✨ Особенности
*   **Гибкая настройка:** выбор длины пароля (от 4 до 30 символов).
*   **Параметры безопасности:** включение/выключение цифр, спецсимволов и букв разного регистра.
*   **Умный фильтр:** возможность исключения неоднозначных символов (типа `l`, `1`, `O`, `0`), которые легко перепутать.
*   **Удобство:** копирование готового пароля в буфер обмена одним кликом.
*   **Современный дизайн:** стильная темная тема благодаря библиотеке `customtkinter`.

## 🛠 Технологии
*   **Python 3.x**
*   **CustomTkinter** (Графический интерфейс)
*   **Pyperclip** (Работа с буфером обмена)
